https://tls.mbed.org/ mbed TLS (formerly known as PolarSSL) makes it trivially easy for developers to include cryptographic and SSL/TLS capabilities in their (embedded) products, facilitating this functionality with a minimal coding footprint.

Binaries for Windows are included compiled from version 2.16.2, other platforms can be built with the sources from https://tls.mbed.org/

The import project will import Windows, Linux and Darwin.
