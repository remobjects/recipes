https://github.com/risoflora/libsagui is a cross-platform library which helps to develop web servers or frameworks. 

Grab a release from https://github.com/risoflora/libsagui/releases and extract it here so that the x86/x86_64 directories are located inside this directory.

the -32 version will import for i386, the -64 version will import for x86_64.
